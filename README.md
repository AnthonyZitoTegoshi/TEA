# TEA
Este repositório contém a página all-in-one da empresa TEA Web Systems.

Trabalho Final de LDI

- Anthony Zito Tegoshi (GU3005771);
- Eric Lima Uno So (GU3006964);
- Thiago Morales Ribeiro (GU3006905);

Nós criamos uma empresa chamada TEA Web Systems e criamos uma parte de contato e feedback, que usa php para se conectar com o banco de dados por meio de classes. O sistema basicamente pega o nome, contato e mensagem do usuário, trata as informações e joga na base de dados para que possamos ver.

Temos o vídeo completo, que inclui a apresentação da empresa e da funcionalidade da parte de contato e feedback, o vídeo com apenas a explicação do código e ainda o código no github e o link para o site na hospedagem.
